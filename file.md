## Design Direction

The interface should feel cinematic and premium rather than corporate.

* **Dark-first visual theme** with charcoal, black, and deep gray backgrounds accented by warm gold or subtle amber highlights.
* **Fullscreen video hero** with smooth scroll-based transitions and layered typography.
* **Immersive project storytelling** where each production unfolds through images, videos, and behind-the-scenes content.
* **Netflix-style content cards** for films and projects with hover previews and metadata.
* **YouTube-inspired testimonial cards** showing video thumbnails, play icons, duration, and creator details.
* **Editorial typography** using large headlines paired with clean body text for a magazine-like reading experience.
* **High-quality motion design**, including parallax imagery, reveal animations, cinematic page transitions, and subtle micro-interactions.
* **Responsive layouts** optimized for desktop, tablet, and mobile while maintaining the premium visual language.
* **Interactive galleries** with filtering, masonry layouts, and lightbox video playback.
* **Consistent branding** that emphasizes craftsmanship, storytelling, and creativity across every page.

This information architecture provides a scalable foundation for Havilah, combining a professional production studio, a creative agency, an educational platform, and a client collaboration portal within a cohesive cinematic experience.


That combination works well for a premium production studio. I would treat them as **two premium identities** rather than simply "dark mode" and "light mode." Both should feel luxurious and cinematic while maintaining the same layout and animations.

# Shared Design Language

Both themes should share:

* Same layouts
* Same animations
* Same typography hierarchy
* Same spacing system
* Same component structure

Only the visual treatment changes.
